デベロッパーゴースト「デベにしき」
-------------------------------------------------------
使い方は簡単です
「デベにしき」がゴーストを更新する様子を眺めるだけです。
-------------------------------------------------------
本体側（デベにしき）をつつくとメニュー

簡単なルール

◯トーク毎に【やる気】が減ります。

◯トーク毎に以下のどれかがランダムで増減します。

【トーク数】
【イベント数】
【切り替え数】
【ユーザー数】
【感想数】

◯メニューの開発頻度を【休む】
に設定すると【やる気】が回復していきます。

◯また、起動時にも大幅に【やる気】が回復します。

※開発頻度の違いによって各ステータスの上昇量に違いはありません。

◯相方（ゴーストにしき）のステータスから
【起動】【終了】【頭なでられ】のトークを追加することができます。

※各１種づつ
