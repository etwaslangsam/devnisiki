■著作意識

◯許可していること
 [1.ゴーストダウンロードページに直接リンクされる]
 [2.ゴーストのネタバレをされる]
 [3.ゴーストに対し批判的な意見を言われる]
 [4.二次創作物の配布をされる(HP素材フリー配布、同人誌販売等)]
 [5.原作の設定と大きくかけ離れた二次創作をされる]
 [6.18禁二次創作をされる]
 [7.追加シェル、バルーン等を配布される]
 [13.配布終了したゴーストをウェブ上で再配布]

×禁止していること
 [11.ゴーストアーカイブへの直リンク]
 [12.公開中ゴーストをウェブ上で再配布]
