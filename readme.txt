デベロッパーゴースト「デベにしき」
-------------------------------------------------------
#各種項目の説明

##やる気
	デベにしきの開発意欲です。  
	ランダムトーク毎に低下します。また、一部の選択肢でも上下します。
	
##開発頻度
	ランダムトークの間隔設定です。  
	**休む（０秒）**にすると、毎分**やる気**が回復します。
	
##開発計画
	ゴーストの上昇させるステータスを決定します。  
	決定した以外のステータスは上昇しません。

##通知設定
	**やる気**が0の時にタスクトレイ通知をするかどうかの設定です。
	ONにすると**やる気**が０の時、ランダムトーク毎に警告が出ます。
